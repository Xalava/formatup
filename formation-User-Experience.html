Introduction à l'User Experience

L’User Experience ou expérience utilisateur est une discipline consistant concevoir les interfaces en se plaçant du point de vue de l'utilisateur.

Objectifs

  • Identifier les points de blocage potentiel dans une interface
  • Concevoir une interface basée sur l’expérience utilisateur
  • Être capable de créer une présentation d’un projet d’interface
