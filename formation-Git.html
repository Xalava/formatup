Git

utilisation et mise en place

Git est un outil distribué de versionnage de code source. Il permet de développer en équipe de façon flexible et agile.

Objectifs

  • Mettre en place le versionnage d’un projet
  • Travailler en équipe sur un projet commun avec Git
  • Garantir la qualité du code
